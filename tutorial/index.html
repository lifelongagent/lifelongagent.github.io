EMNLP 2026 Tutorial
Lifelong Language Agents:
Theory and Practice

October 24–29, 2026
Budapest, Hungary

About The Tutorial

Language agents—LLMs that reason, invoke tools, and interact with users over long horizons—have become a central paradigm in NLP. Yet the field's training and evaluation practices remain stubbornly single-phase: an agent is trained once, scored on a fixed benchmark, and deployed without any mechanism to evolve when tools change, user preferences drift, or new failure modes emerge.

This three-hour cutting-edge tutorial introduces lifelong language agents: agents that learn, align, and evolve across sustained interaction. We organize the area around three coupled questions. Theory: when should an agent think internally versus act externally, and how should computational effort be allocated? We present the Theory of Agent (ToA), a recent framework that formalizes this as an epistemic-effort allocation problem and unifies four canonical failure modes (under/over-thinking, under/over-acting). Practice: we survey concrete recipes spanning capability building, behavioral regulation, and multi-dimensional diagnosis, each paired with contemporaneous external work. Lifelong: we cover continual learning under evolving tools, long-horizon alignment under distribution shift, and agent self-evolution via composed skills and learned world models.

The tutorial provides the NLP community with a unified vocabulary and a field map—not a single lab's research stream—together with classroom-ready pedagogical materials so that the lifelong-agent agenda can diffuse beyond research labs into graduate curricula.



Tutorial Outline

A three-hour tutorial in four parts

Part I · Theory · (50 min)
The Theory of Agent (ToA)

Agent intelligence as an adaptive effort-allocation problem: a knowledge boundary separating what the agent can answer from parameters versus what requires external interaction, an epistemic-necessity criterion for tool use, and an epistemic-effort decomposition whose minimum can only be allocated, never eliminated.

  • ToA basics and the effort decomposition
  • Four canonical failure modes: under/over-thinking and under/over-acting as misallocations of effort
  • Connections to ReAct, Tree of Thoughts, Reflexion, and agentic RL
Part II · Practice · (70 min)
From Theory to Recipes

Concrete recipes organized along three axes that mirror the failure-mode taxonomy, each contrasted with at least one contemporaneous external line so attendees leave with a comparative map.

  • Build — capability construction: high-quality reasoning rationales (Cue-CoT, SRLM), plan-execute-generate frameworks (SAFARI), and RL for tool-integrated reasoning (ToolRL, Search-R2)
  • Regulate — when and how much to think versus act: Self-DC, SMART, AdaCtrl, OTC-PO
  • Diagnose — multi-dimensional evaluation: AppBench, DialogTool, PerLTQA, SafeToolBench, TransBench, compared with GAIA, OSWorld, WebArena, SWE-Bench
Part III · Lifelong · (40 min)
What Changes Without the Single-Phase Assumption

The conceptual core of the tutorial: three thrusts that emerge once an agent is no longer trained once and never updated again.

  • Continual learning under evolving tools, tasks, and users — data-to-policy updates, structured memory abstractions, and skill-library approaches inspired by Voyager
  • Long-term alignment under distribution shift — decomposable preferences that separate personalization from inviolable safety floors, and robust long-horizon defense against prompt-injection and jailbreak chains
  • Agent evolution — composing and creating new tools, internalizing the environment via learned world models, and multi-agent coordination; in ToA terms, progressively collapsing external effort into internal effort
Part IV · Demo + Q&A · 30 min Hands-On Demo, Open Problems, and Discussion

A 15-minute hands-on “diagnose the failure mode” exercise, five open problems exposed by the lifelong framing, and an audience Q&A.
