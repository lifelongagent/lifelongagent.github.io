EMNLP 2026 Tutorial
Lifelong Agents:
Theory and Practice

October 24–29, 2026
Budapest, Hungary

About The Tutorial

Agents are evolving from single-stage task solvers into long-running systems that reason, use tools, maintain memory, collaborate, and interact with users in realistic environments. As tasks, tools, environments, and safety requirements continually shift, sustaining useful and aligned behavior over an agent's lifetime has become a central challenge for NLP.

This three-hour cutting-edge tutorial offers a comprehensive introduction to lifelong agents. We organize representative work around five lifecycle stages: foundations, capability development, behavior and alignment, long-term adaptation, and evaluation. Across these stages, the tutorial covers reasoning and planning, tool learning, retrieval and memory, agentic reinforcement learning, safety, multimodal and embodied agents, multi-agent systems, and long-horizon evaluation.

The goal is a community-facing field map rather than a presentation of any single research line. Attendees will gain a structured overview of the field, a curated reading list, and practical experience in diagnosing agent failures while developing both insight and foresight for lifelong agent systems.

Tutorial Outline

A lifecycle view of agents over sustained interaction

Module 1 · Foundations · 30 min
Foundations of Agents

We establish agents as systems that reason, act, observe, remember, and interact with external environments. The module clarifies tools, environments, memory, trajectories, feedback, and long-horizon interaction.

  • Representative settings: ReAct agents, tool-augmented agents, memory-based agents, multi-agent systems, and embodied agents
  • Diagnostic vocabulary: knowledge boundaries, uncertainty, effort allocation, tool overuse, and tool underuse
  • Opening failure cases that motivate a lifecycle structure for the tutorial
Module 2 · Capability Development · 35 min
Capability Development in Agents

We study how agents acquire capabilities for sustained interaction, organized by the source of capability and by how methods extend agents beyond single-turn generation.

  • Internal reasoning and planning for task decomposition and multi-step execution
  • External interaction through tool invocation, tool composition, retrieval, and API use
  • Experience-based development through memory, session continuity, and feedback-driven improvement
Module 3 · Behavior and Alignment · 40 min
Behavior and Alignment of Agents

We shift from what agents can do to when and how these capabilities should be used. Behavioral control is treated as a central issue in deployed agent systems.

  • Capability selection: when to reason, retrieve, call tools, ask clarification questions, or stop
  • Resource control: reasoning budgets, tool-use budgets, and memory-update decisions
  • Safety constraints: unsafe tool execution, pre-execution safety checks, and deployment-time alignment concerns
Module 4 · Long-term Adaptation · 30 min
Long-term Adaptation of Agents

We extend the discussion from within-session behavior to changes that unfold over time as tools, users, tasks, and safety requirements shift.

  • Memory and experience reuse across sessions
  • Skill acquisition and reusable tool compositions
  • Risks from long-term adaptation: memory pollution, skill drift, self-reinforcing errors, and unsafe capability expansion
Module 5 · Evaluation · 35 min
Evaluation of Agents

We examine how long-running agents should be evaluated beyond final-answer accuracy, then close with a hands-on diagnosis exercise over representative agent trajectories.

  • Task success, cost, latency, and resource use
  • Robustness under distribution shift, tool misuse, unsafe execution, and changing environments
  • Long-horizon properties: memory quality, adaptation, cross-session stability, and degradation

Audience

Who the tutorial is designed for

The tutorial is intended for researchers, Ph.D. students, advanced master's students, and industry practitioners working on language agents, tool learning, RAG, dialogue systems, alignment, AI safety, and agentic system deployment.

The tutorial is self-contained at the conceptual level. Expected background includes familiarity with LLMs, prompting, chain-of-thought reasoning, RAG, and standard fine-tuning or alignment methods such as SFT, DPO, PPO, or RLHF. Basic knowledge of reinforcement learning, Python, or agent frameworks will be useful but is not required.

Materials

Classroom-ready resources for lifelong agents

We will provide classroom-ready slides, an annotated reading list, a hands-on diagnostic exercise, discussion questions for graduate seminars, and a minimal demo notebook.

The reading list will be organized by the tutorial lifecycle: agent foundations, capability development, behavior and alignment, long-term adaptation, and long-horizon evaluation. Materials will be released at least two months before the tutorial through the official website and ACL Anthology.
